# GetDog API

## Описание

Этот проект представляет собой RESTful API для получения информации о различных породах собак, их подпородах и изображениях. API предоставляет возможность выполнения следующих операций:

- Получение списка всех пород собак
- Получение информации о конкретной породе собаки по имени
- Получение списка всех подпород для определенной породы собаки
- Получение информации о конкретной подпороде собаки по имени
- Добавление новой породы собаки
- Удаление породы собаки по имени
- Обновление имени породы собаки
- Добавление новой подпороды для определенной породы собаки
- Удаление подпороды собаки по имени
- Обновление имени подпороды собаки
- Получение случайного изображения собаки для определенной породы и подпороды
- Добавление нового изображения собаки для определенной породы и подпороды
- Удаление изображения собаки по URL
- Обновление URL изображения собаки

## Технологии

Этот проект создан с использованием следующих технологий:

- Java
- Spring Boot
- Spring Data JPA
- MySQL

## Установка и запуск

1. Клонируйте репозиторий на локальную машину:

   ```bash
   git clone https://github.com/llHelltakerll/GetDogApi
   ```

2. Создайте базу данных MySQL и настройте параметры подключения в файле `application.properties`.

3. Соберите и запустите проект с помощью Maven:

   ```bash
   cd GetDogApi
   mvn spring-boot:run
   ```

## Использование

После запуска приложения вы можете использовать следующие эндпоинты для взаимодействия с API:

- GET `/api/breeds/list/all`: Получить список всех пород собак
- GET `/api/breed/{breedName}`: Получить информацию о конкретной породе собаки по имени
- GET `/api/breed/{breedName}/sub_breeds/list`: Получить список всех подпород для определенной породы собаки
- GET `/api/sub_breed/list/all`: Получить список всех подпород собак
- GET `/api/sub_breed/{subBreedName}`: Получить информацию о конкретной подпороде собаки по имени
- POST `/api/breed/add`: Добавить новую породу собаки
- DELETE `/api/breed/delete`: Удалить породу собаки по имени
- PUT `/api/breed/update`: Обновить имя породы собаки
- POST `/api/sub_breed/add`: Добавить новую подпороду для определенной породы собаки
- DELETE `/api/sub_breed/delete`: Удалить подпороду собаки по имени
- PUT `/api/sub_breed/update`: Обновить имя подпороды собаки
- GET `/api/image/{breed}/{subBreed}/random`: Получить случайное изображение собаки для определенной породы и подпороды
- POST `/api/image/add`: Добавить новое изображение собаки для определенной породы и подпороды
- DELETE `/api/image/delete`: Удалить изображение собаки по URL
- PUT `/api/image/update`: Обновить URL изображения собаки
